## **Restosan**

---

## 1. Identitas Proyek

**Nama Proyek:**  
Restosan

**Deskripsi Proyek:**  
Restosan adalah sebuah website yang membantu pengguna menemukan berbagai macam restoran di sekitar mereka berdasarkan kategori dan rating. Website ini juga menyediakan fitur ulasan dan komentar sehingga pengguna dapat membagikan pengalaman serta penilaian mereka terhadap restoran maupun makanan yang telah dikunjungi.

## 2. Teknologi yang Digunakan

- **HTML5**  
  Digunakan sebagai struktur utama halaman website.

- **CSS3**  
  Digunakan untuk mengatur tampilan dan layout website.

- **JavaScript**  
  Digunakan untuk menangani interaksi dinamis seperti filter kategori, rating, komentar, dan manipulasi data.

- **Tailwind CSS**  
  Digunakan untuk mempercepat proses pembuatan tampilan website dengan pendekatan utility-first.

- **JSON**  
  Digunakan sebagai media penyimpanan data (backend dummy) yang ditampilkan pada website.

- **NPM (Node Package Manager)**  
  Digunakan untuk mengelola dependency yang dibutuhkan dalam pengembangan frontend, seperti Tailwind CSS dan Live Server.

- **Node.js**  
  Digunakan untuk menjalankan NPM.

- **Dummy JSON Website**  
  Digunakan untuk mendeploy data JSON agar menghasilkan API endpoint yang dapat diakses oleh website.

---

## 3. Daftar Fitur

- **Login**  
  Digunakan untuk autentikasi pengguna yang telah memiliki akun.

- **Register**  
  Digunakan untuk mendaftarkan akun baru bagi pengguna.

- **Profile**  
  Menampilkan informasi akun pengguna.

- **Dashboard**  
  Menampilkan informasi umum dan rekomendasi restoran.

- **Detail Restoran**  
  Menampilkan informasi lengkap mengenai restoran yang dipilih.

- **Hasil Pencarian**  
  Menampilkan restoran berdasarkan kata kunci atau filter yang dimasukkan pengguna.

- **Review & Rating Restoran**  
  Pengguna dapat memberikan komentar dan penilaian terhadap restoran.

- **Maps**  
  Terhubung dengan Google Maps untuk menampilkan lokasi restoran secara akurat.

- **Navigasi Responsif**  
  Tampilan website menyesuaikan berbagai ukuran perangkat (mobile, tablet, dan desktop).
